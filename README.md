# SmartTax Pro

A personal finance and tax management system.

## Features

- Income Tracking
- Expense Tracking
- Tax Calculation
- Savings Goals
- Analytics
- PDF Reports

## Project Structure

financial_engine.py
savings_engine.py
analytics_reporting_engine.py
main.py

## Installation

pip install -r requirements.txt

## Run

python main.py

## Architecture

Financial Engine
- Income
- Expenses
- Taxes
- Balance

Savings Engine
- Goals
- Progress

Analytics & Reporting Engine
- Analytics
- Reports

Main Controller
- Connects all modules
